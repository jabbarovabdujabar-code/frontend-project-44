## Usage

1. Установите пакет глобально (или через npm link):

```bash
npm install -g .
```

2. Запустите игру:

```bash
brain-even
```

3. Пример игры (победа):

- Question: 8 -> yes
- Correct!
- ... (3 раза)
- Congratulations, <name>!

4. Пример ошибки:

- Question: 15 -> yes
- 'yes' is wrong answer ;(. Correct answer was 'no'.
- Let's try again, <name>!
